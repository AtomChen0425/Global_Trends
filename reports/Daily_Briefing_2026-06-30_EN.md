### 1. [Open-Vocabulary and Referring Segmentation for 3D Gaussians Using 2D Detectors](https://arxiv.org/abs/2606.30638v1)
👤 **Authors:** Jameel Hassan, Yasiru Ranasinghe, Vishal Patel
<details>
<summary><strong>📄 Paper Summary:</strong> **Background**

3D Gaussian Splatting (3DGS) is a powerful technique for 3D scene reconstr...</summary>

**Background**

3D Gaussian Splatting (3DGS) is a powerful technique for 3D scene reconstruction. Extending 3DGS with language understanding is crucial for applications like embodied AI. Prior methods often rely on distilling CLIP features into the 3D scene representation. However, these approaches face limitations: instance grouping can be rigid (requiring a fixed number of instances) or noisy, and CLIP's reliance on noun phrases restricts semantic understanding, hindering complex spatial reasoning and referential expression grounding.

**Technical Implementation**

GaussDet addresses these limitations by moving away from dense CLIP feature distillation. Instead, it utilizes discrete, open-vocabulary 2D object detectors with referring expression capabilities. The core innovation lies in learning instance features for individual 3D Gaussians, enabling the decomposition of the scene into distinct 3D instance groups. A novel View-Aggregated Semantic Label Distribution (VASD) is generated for each 3D instance. This is achieved by rendering these instance groups and aggregating semantic votes from multi-view 2D detections. This view-aggregation process acts as a strong regularizer, effectively mitigating noise and spurious labels that can arise from imperfect instance grouping.

**Application Scenarios**

GaussDet's architecture facilitates a direct, zero-shot extension from simple language queries to complex referential expression grounding. This capability is demonstrated through evaluations on two key tasks: open-vocabulary segmentation (LeRF-OVS, ScanNet) and referring expression grounding (Ref-LeRF). The method shows consistent improvements over existing approaches, notably achieving a significant 16.7% mIoU improvement in referential grounding within a strict zero-shot setting. This indicates its potential for more intuitive and precise human-robot interaction and scene understanding.

**Summary**

GaussDet represents a significant advancement in language-driven 3D scene understanding by overcoming the limitations of previous CLIP-based methods. By leveraging discrete 2D object detectors and a novel view-aggregation strategy for semantic labeling, it achieves robust instance decomposition and complex referential grounding with zero-shot capabilities. The method's strong performance on key benchmarks highlights its practical utility for real-world applications requiring sophisticated scene interpretation.

Although existing benchmarks have advanced video understanding evaluation across multiple dimensions, they provide limited visibility into how models maintain world state over time. We propose SVCBench, a Streaming Video Counting Benchmark that repositions counting as a minimal, controlled probe for diagnosing models' world-state maintenance capability. We decompose this capability into object counting and event counting, forming 8 fine-grained subcategories.
